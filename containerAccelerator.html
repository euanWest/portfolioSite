Container Accelerator

Task by an idustry client, KMPG, to make a container accelerator that their developers could use to quickly deploy cloud based infrastructure to Amazon Web Services. The project was carried out in a team using agile methodologies like Kanban and Scrum.

Our solution used Terraform to deploy and destroy customizable kubernetes clusters.

Customization

The clusters to be deployed could be customized using the included tfvars file. Users could configure their clusters name, region, name of their VPC, private and public subnets and more.

User Test

We held a user test that aimed to garner qualitive feed back, as we knew we would only get a small pool of users. This proved invaluable as the expertise of our user, as they were professionals, meant our final solution was strong as a result of implementing their feedback.
