Honours Project

The aim of this project was to investigate telemetry data or other similar data from motorsports for use in a videogame. The is development of this project was split into three parts that were worked on iteratively. The product of the project is a prototype racing game made with Unity, where the player races against ghost cars generated from the position data from Formula 1 cars.

Telemetry Data Server

A Python Flask server hosted on a google cloud compute instance is used to serve telemetry data as a json. FastF1, a python library mainly used for analytics, is used to source telemetry data.

Implementation of Telemetry Data

Once the telemetry data json is downloaded it is parsed into a list of drivers, each driver has a number and a list of segments of data. Segments of data include a time and position data for where the driver was at that time. Lerp functions were used to move the ghost cars between the segments of data.

An editor tool was made to help align the track asset with the position data of the F1 cars. The editor tool instanstiates a prefab for every segment of data in the Json file downloaded from the server.

Player Car Controller

There is a very basic car controller that the player can use to race against the ghosts generated form the Formula 1 data.


Track asset is from Assetto Corsa developed and published by KUNOS-Simulazioni Srl.

Car asset is by chwashere123.
